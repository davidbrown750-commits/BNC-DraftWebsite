B.2 Daily Function Check (Start of Shift)

B.2 Daily Function Check (Start of Shift)
  • Unit powers on; no error codes
  • Battery ≥ 80% (or unit on charge)
  • Date/time correct
  • GPS lock obtained (outdoor)
  • Self-test passed
  • Energy calibration check (Cs-137 photopeak at 662 keV ± 1%)
  • Audible alarm tested
  • Visible alarm tested
  • Vibration alarm tested (if equipped)
  • Connectivity tested (Wi-Fi/cell)
  • Test transmission to reachback succeeds
  • Background spectrum acquired and stored
  • Spectrum store has space for at least 50 events
  • Operator log signed

B.3 Weekly Function Check

B.3 Weekly Function Check
  • All daily items
  • Check source verification (Cs-137 and Co-60 if available)

- Cs-137 photopeak at 662 keV

- Co-60 photopeaks at 1173 and 1333 keV

  • Resolution check (FWHM at 662 keV within spec)
  • Battery rotation: oldest battery cycled out
  • Charger cables inspected
  • Carry case inspected (no contamination, all accessories present)
  • Firmware version recorded
  • Spectra archived to records server
  • Maintenance log entry signed

B.4 Post-Use Check (After Any Real Event)

B.4 Post-Use Check (After Any Real Event)
  • All event spectra saved and exported
  • Photos transferred to records
  • Reachback worksheet completed
  • Unit external surfaces inspected for contamination (survey meter)
  • Decontamination performed if needed (per agency procedure)
  • Battery rotated to charger
  • Storage cradle/case dried and clean
  • Defects logged (button stiff, screen scratch, etc.)
  • Unit returned to ready status
