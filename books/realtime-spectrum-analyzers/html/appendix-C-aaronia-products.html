Cascading capability: four V6 PLUS units yield 1 GHz aggregate RTBW.

Common features across the family:

  • Sweep speed: 300 GHz/s (440 GHz/s with upgrade), 1+ THz/s class.
  • POI: 10 nanoseconds (with hardware peak detection).
  • I/Q recording: 24/7 streaming up to 245 MHz bandwidth.
  • Software: RTSA Suite PRO (Windows, Linux, macOS).
  • Connectivity: USB-C, Ethernet, 10 MHz reference I/O.

RTSA Suite PRO Module Matrix

  • Spectrum analysis modes: live trace, waterfall, persistence, density, wrapped spectrum.
  • Modulation analysis: CW, sweep, noise, pulse, FSK, BPSK, ASK, QPSK, QPSK-C, 16-QAM, 64-QAM, 256-QAM, 1024-QAM, 4096-QAM, OFDM, raster image, echo/reflection.
  • Direction finding: real-time 3D bearing with elevation, cross-bearings, mobile truck support, GPS position solving, 3D map view.
  • Recording: 24/7 I/Q streaming up to 245 MHz, SigMF format, replay-as-live.
  • Triggers: frequency mask, level, pulse, external, software-defined.
  • Visual programming: block-based pipeline editor with custom Python and C++ blocks.
  • Data export: SigMF, KML, KMZ, GeoJSON, NMEA, CSV, MQTT, REST API.

Accessory Ecosystem

  • Antennas: HyperLOG 60100/60200/60300/60400/60600 (broadband), OmniLOG 30800/70600 (omnidirectional), MagneticLOG (low-frequency loop), specialized horns and patches.
  • Reference oscillators: external 10 MHz input compatible with rubidium, GPSDO, OCXO sources.
  • Cables: calibrated low-loss test cables, semi-rigid and conformable options.
  • Mounting: mast adapters, rack-mount kits, tripod heads, magnetic mounts.
  • Calibration: factory recalibration services, NIST/PTB-traceable.
